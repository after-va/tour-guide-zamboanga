🚀 HOW TO USE
═══════════════════════════════════════════════════════════════════

1. START WEB SERVER:
   - Open XAMPP Control Panel
   - Start Apache
   - (Optional) Start MySQL for future backend

2. ACCESS WEBSITE:
   - Open browser
   - Go to: http://localhost/tour-guide-zamboanga/

3. EXPLORE AS TOURIST:
   - Click "Browse Tour Guides"
   - View guide profiles
   - Click "Book Now" on any guide
   - Login/Register as tourist
   - Explore dashboard

4. EXPLORE AS GUIDE:
   - Click "Login" → "Tour Guide Login"
   - Click "Apply Now" to register
   - Fill guide application form
   - Login and explore guide dashboard
