Quick Start

  1. Import tourguidesystem.sql to create database structure
  2. Import tourguidesystem-data.sql to add country codes
  3. Run test-operations.php to verify setup
  4. Login to Admin Portal with:
    • Username: admin@tourismozamboanga.com
    • Password: admin123
  5. Create tour spots, packages, and schedules
  6. Register as Tourist or Guide
  7. Test booking functionality
